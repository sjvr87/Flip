Flip dev — how to start (Windows)
=================================

BEST: double-click flip-dev.bat in the Flip repo folder:
  C:\Users\tomas\Documents\Flip\flip-dev.bat

PowerShell (ONE line — copy only the line below, not "PS C:\...>"):
  powershell -NoProfile -ExecutionPolicy Bypass -File "C:\Users\tomas\Documents\Flip\scripts\Start-FlipDev.ps1"

From repo root (npm.cmd avoids PowerShell blocking npm.ps1):
  cd /d C:\Users\tomas\Documents\Flip
  npm.cmd run flip:dev

cmd.exe (3 lines):
  cd /d C:\Users\tomas\Documents\Flip
  npm.cmd run dev:connect:restart

Or from anywhere:
  C:\Users\tomas\Documents\Flip\scripts\flip-dev.cmd

NEVER paste a PowerShell prompt into the terminal (lines starting with "PS C:\...>").

NEVER use bare "npm" in PowerShell — use npm.cmd or the scripts above.

flip-dev.bat closes on success (Metro stays in its own window).
Metro window: taskbar -> Command Prompt titled "Flip Metro" (bundler log, QR, exp://LAN:8081).
The flip-dev.bat window is only the connect script - not Metro. Close it after "Done" if you like.
Use flip-dev-pause.bat if you want this window to stay open to read the log.
